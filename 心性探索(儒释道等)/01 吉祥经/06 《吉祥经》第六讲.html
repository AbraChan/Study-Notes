06 《吉祥经》第六讲

《吉祥经》第六讲

“奉养父母亲，爱护妻与子，从业要无害，是为最吉祥”



【经论引用】
  • 在《善生经》中佛陀这样讲：“夫为人子，当以五事敬顺父母，云何为五? 一者供奉能使无乏， 二者凡有所为先白父母， 三者父母所为恭顺不逆， 四者父母正令不敢违背， 五者不断父母所为正业。”
  • 同时佛陀也讲到父母该怎么样对待子女，也是五点：“父母复以五事敬亲其子。云何为五？一者制子不听为恶，二者指授示其善处，三者慈爱入骨彻髓，四者为子求善婚娶， 五者随时供给所需。”
  • 《善生经》里也讲到一个丈夫应该怎样对待自己的太太：“夫之敬妻亦有五事。云何为五？一者相待以礼， 二者威严不缺， 三者衣食随时， 四者庄严以时， 五者委付家内。”
  • “善生，夫以此五事敬待于妻。妻复以五事恭敬于夫。云何为五？一者先起，二者后坐，三者和言，四者敬顺，五者先意承旨。”

  • 《净行品》中说：菩萨在家，当愿众生，知家性空，免其逼迫…妻子集会，当愿众生，怨亲平等，永离贪着。”。。。佛曾对一个发心出家的人说了个偈子：流转三界中，恩爱不能脱，弃恩入无为，真实报恩者。

  • 《法句经》的第一偈：“诸法意先导，意主意造作。若以染污意，或语或行业，是则苦随彼，如轮随兽足。若以清净意，或语或行业，是则乐随彼，如影不离形。” 

  • 佛在舍卫国，时有一梵志，从罗越只国来，欲得学问，便到舍卫国。见父子二人耕田，毒蛇螫杀其子，其父犹耕如故，不看其子，亦不啼哭。梵志问曰：“此是谁儿？”耕者答曰：“是我之子。”曰：“是卿之子，何不啼哭？”耕者答曰：“人生有死，夫盛有衰，善者有报，恶者有对，愁忧啼哭，无益死者。卿今入城，我家某处，愿过语之，吾子已死，持一人食来。”梵志自念，此是何人，而无返复，儿死在地，情不愁忧，反更索食，此人不慈，无复与比。
  • 梵志入城，诣耕者家，见死儿母，即便语之：“卿儿已死，其父寄信，持一人食来。”母答曰：“可。”即便操作，面无戚容梵志曰：“何以不念子耶？”儿母即为梵志说譬喻言：“儿来托生，我亦不呼，儿今自去，非我能留。譬如行客，因过主人，客人自去，何能得留？我之母子，亦复如是。去来进止，非我之力，随其本行，不能救护。”
  • 复语其姊：“卿弟已死，何不啼哭？”姊复说譬喻向梵志言：“譬如巧师，入山斫木，缚作大筏，安置水中。卒逢大风，吹破筏散，随水流去，前后分张，不相顾望，我弟亦尔，因缘和合，共一家生，随命长短，死生无常，会合有离，我弟命尽，各自所随，不能救护。”复语其妇：“卿夫已死，何不啼哭？”妇说喻向梵志言：“譬如飞鸟，暮宿高树，同止共宿，伺明早起，各自飞去，行求饮食。有缘即合，无缘即离，我等夫妇亦复如是。无常对至，随其本行，不能救护。”复语其奴：“大家已死，何不啼哭？”奴复说譬喻言：“我之大家，因缘和合。我如犊子，随逐大牛，人杀大牛，犊子在边，不能救护大牛之命，忧愁啼哭，无所补益。”梵志闻已，心感自责，不识东西，我闻此国孝顺奉事恭敬三宝，故从远来，欲得学问，既来到此，了无所益，更问行人，佛在何许？欲往问之。
  • 行人答言：“近在只桓精舍。”梵志即到佛所，稽首作礼，却坐一面，合掌低头，默无所说。佛知其意，谓梵志曰：“何以低头不乐？”梵志曰：“所愿不果，违我本心，是故不乐。”佛语梵志：“有何所失，愁忧不乐？”梵志对曰：“我从罗越只国来，欲得学问，既来到此，见五无返复。”佛问梵志：“何等五无返复？”梵志曰：“我见父子二人耕田下种，儿死在地，父亦不愁，居家大小都无愁悲，是为大逆。”佛言：“不然，不如卿语，此之五人最为返复，知身非常，身非己有，往古圣人不免斯患，何为凡夫大啼小哭，无益死者，世俗之人，无所识知，生死流转，无有休息。”梵志心开意解，说道：“如病得愈，盲者得视，如暗得明。”于是即得道迹。（参考《经集.3大品.8箭经》）
  • 上面这个故事出自《五无返复经》。返复有恩情酬报的意思。



【其它摘抄】
  • 这是《善生经》中讲到的父母跟子女双方的责任与义务。佛陀是非常中肯的，他并没有偏向任何一方。而是强调亲子双方都要尽到自己的责任和义务。人跟人之间，无论是谁，都是关系，关系就是双方的互动，作用力与反作用力的原理，所以在一种关系里，必然双方都有自己该尽的责任和义务。

  • 作为修行者，我当然深知家庭之乐是轮回的枷锁之一。但是，仍然真心希望你们每一个人都拥有幸福的家庭、幸福的人生。幸福与修行并不是不可兼容的，建设幸福的过程本身就是一种不简单的修行。想得到幸福并不等于懂得如何获得幸福，也不等于有能力获得幸福，需要修炼。

  • 我们可以建设幸福体验幸福，但是不要执着，心里要明了世事终究是无常的，随时能够接受无常的存在。《净行品》中说：菩萨在家，当愿众生，知家性空，免其逼迫…妻子集会，当愿众生，怨亲平等，永离贪着。”不贪着就是看破，看破世界上的万事万物都是因缘和合而成。过去的因，加上现在的缘，聚集成一个家庭。缘尽的时候，自然也就离散了。佛曾对一个发心出家的人说了个偈子：流转三界中，恩爱不能脱，弃恩入无为，真实报恩者。

  • 有时候，夫妻中的一方会瞧不起另外一方，觉得对方配不上自己，觉得自己屈就了。事实上，从轮回和业果的角度来说，当下你所拥有的就是你最合适的匹配，你应得的果报，没有什么好挑剔的。

  • “从业要无害”。一个人在社会上成家立业，找工作做事业，要做对社会对人类有益的工作，佛教里称这个为“正命”，就是以正确的方式维持生计，所从事的职业对自己对他人对众生的身心要无害。偈子里所讲的“无害”是指符合法律符合道德规范的职业，如果对一个佛教徒来说，符合道德和法律的标准还不够，因为不一定如法，不一定遵守戒律，不一定利于我们的修行，不一定有助于我们的解脱。正命是八正道的道支之一。正命的反面就是邪命 ，戒律有要求出家人不能做四种邪命。因为按照印度原始佛教，出家人的本分是托钵乞食。“邪”就是知见不正、思想不正。谋生的方法手段不正，这叫“邪命”。哪四种邪命呢?
    • 第一种叫“下口食”, “下口食”就是自己耕种，种田或者是种菜  ...  佛对于出家人要求的是乞食，所谓“日中一食，树下一宿”... 少欲知足，身心安稳、定慧增长，所以佛取这种方式。
    • 第二种叫“仰口食”，“仰口食”是观察星象，为人说吉凶祸福。
    • 第三种叫“方口食”， “方”是讲四方，“口”是说交际应酬。天天到外面去交际应酬，结交权贵、拉拢信徒、得到供养，用这种方法来生活的，统统叫做“方口食”。 ...  心地不清净，道心都失掉了！
    • 第四种叫“维口食”，“维口食”是以咒术，画符念咒，或者是给人治病，或者是占卜吉凶，这里面包括现在所讲的算命、看相、看风水，这些对于出家人都是不应当做的！

  • 另外在其它的经里也讲到有几种邪命也是出家人所不做的:
    • 第一种是以欺诈表现奇特之异相，以欺诈方式表现自己有特殊功能，比如有天眼通、有神通可以从空中取药等等，但都是假的;
    • 第二种是夸耀自己功德，通过夸耀自己的功德以引起对方的恭敬然后带来供养。正信的出家人一般是不说自己怎样怎样的;
    • 第三种叫“高声威吓”，比如一些人，穿上出家人的衣服，假扮成出家人，挨门挨户去化缘。有的会恐吓说你要出事你家要出什么事啦，让你心里升起恐慌，然后他就可以骗钱了。
    • 第四种就是称赞供养者,以此来获得供养。比如说，现在也许正缺一些衣服，然后对居士说“供养衣服功德特大，以后能够富贵”等等，引起对方的注意然后得到供养。

  • 在《梵网经》里就讲“菩萨不能沽酒”，意思是不能买卖酒，因为买卖酒会使很多众生受到损害。因此使自己也受到损害，所以菩萨戒把沽酒戒放在前面，不饮酒戒放在后面。

  • 佛有五眼六通，了知一切有情的宿世因缘，了知父子夫妇兄弟等眷属的遇合，皆由宿世因缘而起。而这种因缘，又是属于怨怼的多，属于欢爱的少。凡对于子女等一切所爱的人，终身作牛马，牺牲自己，皆因宿世欠债，今生须还债的缘故。亲人眷属多由这种因缘而遇合，可是凡夫烦恼缠缚，欠人者常图逃赖，人欠者不肯舍弃。我们要一方面知道眷属缘成于孽缘，另外一方面又必须知怨亲平等。而欠债当还，被欠当舍，故在家则仍能调和眷属，无损亲情；出家又即能割断爱缠，无所留恋。而眷属死亡之时，除替他念佛说法，作实际有益之事以外，也自然不作无谓的悲啼了。经文里面的梵志认为人死，他的眷属必须悲伤啼哭，这是凡夫俗见。这个农人的家属知道缘尽则离，不需要悲哀留恋，这是符合佛陀开示的正知见的。人死后有升有沉，根据个人善恶业而不同，忏悔与念佛，都可以灭罪而免堕落；而临死时之一念，也就是是临终最后一个心路过程，决定了来世的去向，是上升还是下堕，全在这最后时刻。详细的解说，大家可以听《阿毗达摩》离心路过程这一课。死者要往生的时候，家属应当为他说法或高声念佛，以正死者之心念。如果悲号呼唤，只会增长死者对世间的迷恋而导致死者的堕落，这不是爱死者，反而是害他了。我们学佛的人一定要明白这个道理，明白怎样做是真正对自己的亲人有益的。



【思考题】
一、在家人第一重要的修行是什么？
  • 奉养父母亲爱护妻子（或者丈夫）与孩子，作为在家人来说这是我们第一重要的修行。



二、子女如何恭敬、孝顺父母? 你做到了哪些？
  • 在《善生经》中佛陀这样讲：“夫为人子当以五事敬顺父母，云何为五? 一者供奉能使无乏， 二者凡有所为先白父母， 三者父母所为恭顺不逆， 四者父母正令不敢违背， 五者不断父母所为正业。”
    1. 物质上要供养父母，使父母不缺乏，不缺衣不少食;
    2. 做什么事先跟父母商量，表示对父母要尊敬，另外父母的岁数大生活经验丰富，也会给我们提很多有用的建议;
    3. 父母的所作所为不要随意地背叛、批评;
    4. 父母的命令，父母叫我们做的事，不要违抗;
    5. 把父母的事业继承发扬下去。这是指父母所做的正业，邪业就不行了，比如杀猪宰牛，这种工作子女没有必要继承，你可以换工作。



三、父母如何对待子女？ 你做到了哪些？
  • 父母该怎么样对待子女，也是五点：“父母复以五事敬亲其子。云何为五？一者制子不听为恶，二者指授示其善处，三者慈爱入骨彻髓，四者为子求善婚娶， 五者随时供给所需。”
    1. 要教育自己的子女、要约束自己的子女，不要让他们做坏事;
    2. 在生活中告诉他什么可以做什么不能做，哪些事情是好事;
    3. 父母对子女非常慈爱，慈爱到这种爱心到了骨髓里去了，我们讲，父慈子孝，这是有顺序的，首先父母要对子女付出慈爱，有爱滋养的子女才懂得孝顺父母。父母是子女的第一老师、榜样;
    4. 要为子女找到好的婚配，女的要找好的婆家，男的找好的媳妇。
    5. 子女上学、成家立业需要本钱，父母要在经济上给予支持。



四、夫妻之间互相应该如何对待？核心精神是什么？你做到了哪些？
  • 《善生经》里也讲到一个丈夫应该怎样对待自己的太太：“夫之敬妻亦有五事。云何为五？一者相待以礼， 二者威严不缺， 三者衣食随时， 四者庄严以时， 五者委付家内。”
    1. 以礼对待妻子，夫妻之间有分寸的，夫妻之间显然是要亲爱，要相亲相爱，但是不能没有界限、不能过于狎昵，否则就完全没有了章法、没有界限了。
      • 有的丈夫把妻子当成自己的所有物看待的，不是把她当成一个人，当成一个独立的人，当成自己的所有物当然也就不会相待以礼，这更是不恰当的。
      • 夫妻之间，有恩，也很容易生怨。为什么？因为太亲密，说话就不客气，不留余地了。也因为太了解，说的话常常一针见血击中对方痛处。不小心，说出来的一句两句，就深深伤害对方的自尊了。这一句两句并不是说完就过去了的，它是留在对方心里日久弥深，为未来的种种冲突矛盾分裂留下诱因的。
      • 以中国传统的观点，夫妻要相敬如宾、举案齐眉。
    2. 指作丈夫的在妻子面前不能太随便，不能倒架子。在妻子面前没有威严，容易让妻子变得放肆，变得目中无人。这也不利于夫妻正常关系。
    3. 丈夫有责任保证妻子的物质生活不缺乏，男人养家养妻子孩子，这是责任和义务，也是一个男人的能力。
    4. 作丈夫的不仅要从衣食上要保证妻子有房子住有衣穿有饭吃，而且还要让妻子有合适的装饰品，打扮得庄严得体,出门见得了人，不寒碜。
    5. 这有两个意思，一是把家庭的管理权力交给妻子，男主外女主内。在一个家里，两个人分工合作，发挥各自的长项，才能更好地经营家庭。另外一个意思是信任妻子，把家庭财产交给妻子掌管，而不是自己另外建个小金库。
      • 丈夫是妻子的“官星”，妻子是丈夫的“财星”，所谓“妻财”，以妻为财。一个男人如果不懂得打理好自己的财星，是极其不明智的。很多成功的男人都是“妻管严”，他们并不是怕自己的妻子，而是懂得如何处理好与妻子的关系，抓大放小，在日常琐事上谦让妻子，和谐夫妻关系。妻子对自己家庭财产应有效地控制并保持收支平衡，使收大于支，合理地使用金钱。
      • 对于金钱的安排，佛陀讲了四点：1 、用于日常生活，提高生活水平；2 、再投资；3 、供养出家人，行慈善事业；4 、储存起来，以备急用。
  • “善生，夫以此五事敬待于妻。妻复以五事恭敬于夫。云何为五？一者先起，二者后坐，三者和言，四者敬顺，五者先意承旨。”
    1. 早上要比自己的丈夫起得早。——  做个勤劳的主妇；
    2. 先让丈夫安坐下来了，自己才落座。——  尊重丈夫一家之主的地位;
    3. 和言的意思是讲话要和气声音不要大。在家庭里面要和言爱语。态度要温柔，嗓门不要大。不要动不动就呵斥丈夫，嫌弃丈夫这不好那不好，没本事没能力。好男人是好女人塑造出来的。好丈夫好孩子都是恰当的肯定和鼓励培养出来的。——  在言行上恭敬丈夫；
    4. “敬顺”和“和言”是接近的，敬顺是讲心态上。——  在心态上恭敬丈夫；
    5. 意思是说丈夫有一些意思还没有说出来，作妻子的就知道了，就马上就去落实和照办。在他开口以前，就明白了他的意思。其实这是两人之间的心意相通。——  对方的意思还没有讲就已经明白去照办，做个贴心的女人。
  • 核心精神是互爱、互礼、互敬。



五、佛陀对于家庭金钱的安排是如何教导的？
  • 一是把家庭的管理权力交给妻子，男主外女主内。信任妻子，把家庭财产交给妻子掌管，而不是自己另外建个小金库。
  • 对于金钱的安排，佛陀讲了四点：1 、用于日常生活，提高生活水平；2 、再投资；3 、供养出家人，行慈善事业；4 、储存起来，以备急用。



六、梵志在舍卫国的经历给您什么启示？
  • 佛有五眼六通，了知一切有情的宿世因缘，了知父子夫妇兄弟等眷属的遇合，皆由宿世因缘而起。而这种因缘，又是属于怨怼的多，属于欢爱的少。凡对于子女等一切所爱的人，终身作牛马，牺牲自己，皆因宿世欠债，今生须还债的缘故。
  • 亲人眷属多由这种因缘而遇合，可是凡夫烦恼缠缚，欠人者常图逃赖，人欠者不肯舍弃。我们要一方面知道眷属缘成于孽缘，另外一方面又必须知怨亲平等。而欠债当还，被欠当舍，故在家则仍能调和眷属，无损亲情；出家又即能割断爱缠，无所留恋。而眷属死亡之时，除替他念佛说法，作实际有益之事以外，也自然不作无谓的悲啼了。经文里面的梵志认为人死，他的眷属必须悲伤啼哭，这是凡夫俗见。这个农人的家属知道缘尽则离，不需要悲哀留恋，这是符合佛陀开示的正知见的。人死后有升有沉，根据个人善恶业而不同，忏悔与念佛，都可以灭罪而免堕落；而临死时之一念，也就是是临终最后一个心路过程，决定了来世的去向，是上升还是下堕，全在这最后时刻。
  • 死者要往生的时候，家属应当为他说法或高声念佛，以正死者之心念。如果悲号呼唤，只会增长死者对世间的迷恋而导致死者的堕落，这不是爱死者，反而是害他了。我们学佛的人一定要明白这个道理，明白怎样做是真正对自己的亲人有益的。
  • 不管我们与家人眷属的缘分是成于孽缘还是善缘，今生能够成为一家人，首先就应该惜缘，以《善生经》中佛陀开示善生所说的子女对待父母、父母对待子女、丈夫对待妻子、妻子对待丈夫的方式，去对待对方。
  • 其次，要明白怨亲平等，不要有怨与亲的分别，也不要有欠和被欠的分别。欠债当还、被欠当舍，切莫欠人者逃赖，人欠者不舍。
  • 三者，了知世间万事万物都是因缘和合而成，缘来了我们要惜缘，缘尽了，就让它去，不必过于喜怒哀乐，不要执着。当我们失去身边最亲近的人的时候，如果做不到不悲伤哭啼，那就允许让自己悲伤哭啼，因为这样一份情感陪伴了我们这么久、又这么深，它要走，总归会有一个过程，就比如惯性这种东西，是确确实实存在的，而这个过程，也是确确实实会存在的，只不过对于不同的人来说，过程所持续的时间长短、曲折程度不一罢了。所以可以允许有一个过程，该有就让它有，该来就让它来，不要拒绝，不要反抗。过程的长短曲折让它自自然然地、适合自己就好，但是要知道有一天这个过程会结束，会走掉，而我们也会让它走，或者自己走出来，放下它。而不要一直沉溺，也不要执着。物来则应、物去不留。
  • 面对亲人临终，尽量不要悲号呼唤，而是做一些让临终的亲人产生正念、心境平和的事情，比如为临终的亲人说法或高声念佛。

  • 弘逸师兄补充庄子妻子过世的故事。



七、如何以五戒的精神选择职业？
  • 对于在家居士来说，要从业无害，可以以五戒的精神来衡量。正确选定自己的职业方向、工作方向。所从事的职业对自己对他人对众生的身心要无害，这是最吉祥的。
    1. 第一不杀生，凡是与杀生有关的职业不要做，已经做了想办法转、想办法调整。
    2. 第二不偷盗，现在做生意有时会偷税漏税，那也是偷盗，我们佛教徒在社会上一定不要偷税漏税，那是偷国家的钱，偷一国众生的钱。
    3. 第三不邪淫，我们在家居士从业一定不能够开象一些隐藏有色情服务的桑拿洗浴中心、发廊、按摩馆那样的行业，这些行业一定不要做。如果做了马上停下来改行，假如我们所做的职业所从事的行业有色情服务或者会诱导别人犯这些错误，对自己的损害是非常之大。不仅仅是损害自己，而且损害自己的后代子孙。包括开网吧开迪厅，容易出现污七八糟的事，所以这些职业佛教徒一定要远离，很损自己的阴德。
    4. 第四个就是不妄语，说诚实语。对于中介、律师、会计这些职业，要小心。
    5. 第五个不饮酒，这个职业就比较多了，包括卖酒。
      • 在《梵网经》里就讲“菩萨不能沽酒”，意思是不能买卖酒，因为买卖酒会使很多众生受到损害。因此使自己也受到损害，所以菩萨戒把沽酒戒放在前面，不饮酒戒放在后面。
  • 《菩萨戒经》讲到在家五邪命，一制卖肉，二，制卖武器乃至鱼网类，三制卖烟酒一切麻醉品，四，制毒，五制卖出售众生。这些都是不宜从事的行业，其实也囊括在五戒的范畴里。
